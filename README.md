# Instagram-Clone
A clone of a social media platform called Instagram.<br > 

# Application stack 
MongoDB<br >
Express Js<br > 
React Js<br >
Node Js.

Login<br > My Profile<br > Follow or Unfollow other users<br > CRUD Posts<br > Add comment<br >Like & unlike post<br > Logout
